# My_Games
게임을 개별로 저장해서 처리하기 위한 장치.

GoLang을 이용해서 만드는 중.

Gemini와 자체 GPT-OSS:20을 이용해서 만드는 중.

# 실행 시
go run -mod=vendor .

# 빌드 시 (exe 파일 생성)
go build -mod=vendor -o pixel_game.exe .
